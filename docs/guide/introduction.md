# 项目介绍

Nest-Admin-Soybean 是一个基于 NestJS + Vue3 + Prisma 构建的企业级全栈管理系统，提供开箱即用的解决方案。

## 项目特点

### 🏢 多租户架构

系统内置完善的多租户数据隔离机制，支持 SaaS 模式运营：

- **自动租户过滤**: 所有数据库查询自动按租户隔离
- **租户上下文**: 基于请求头自动识别租户身份
- **灵活配置**: 支持跳过租户过滤的特殊场景

超级管理员租户 ID 为 `000000`，具有跨租户管理权限。

### 🔐 完善的权限系统

基于 RBAC (Role-Based Access Control) 的权限控制系统：

- **多层级权限**: 支持菜单、按钮级别的细粒度权限控制
- **角色管理**: 灵活的角色定义和权限分配
- **部门管理**: 支持组织架构的部门权限隔离
- **数据权限**: 支持全部、自定义、本部门、本人等多种数据范围

### 🔒 安全加密

前后端数据传输采用 AES + RSA 混合加密：

1. 前端生成 AES 密钥，使用 AES-CBC 加密数据
2. 使用服务端 RSA 公钥加密 AES 密钥
3. 后端使用 RSA 私钥解密 AES 密钥，再解密数据

敏感操作（登录、修改密码等）自动启用加密传输。

### 📊 日志监控

完整的日志和监控体系：

- **结构化日志**: 基于 Pino 的高性能日志系统
- **操作审计**: 自动记录用户操作日志
- **登录日志**: 完整的登录历史和安全审计
- **健康检查**: 支持 K8s liveness 和 readiness 探针
- **性能监控**: Prometheus 指标暴露

### 🎭 演示账户

内置安全可靠的演示账户系统：

- 52 个只读权限，可查看所有模块
- 自动拦截所有写操作
- 适合产品演示和功能体验

## 核心功能

### 系统管理

- **用户管理**: 用户的增删改查、角色分配、密码重置
- **角色管理**: 角色定义、权限分配、数据范围设置
- **菜单管理**: 动态菜单配置、权限关联
- **部门管理**: 组织架构管理、部门权限
- **岗位管理**: 岗位定义和管理
- **字典管理**: 数据字典的统一管理
- **参数配置**: 系统参数的动态配置
- **通知公告**: 系统通知和公告发布

### 系统监控

- **在线用户**: 实时在线用户管理、强制下线
- **操作日志**: 完整的操作审计记录
- **登录日志**: 登录历史和安全审计
- **定时任务**: 任务的定时执行和管理
- **健康检查**: 系统健康状态监控
- **性能指标**: Prometheus 性能指标

### 租户管理

- **租户列表**: 租户的增删改查
- **租户套餐**: 套餐定义和管理
- **租户隔离**: 数据自动隔离和权限控制

### 文件管理

- **文件上传**: 支持多种文件类型上传
- **缩略图**: 图片自动生成缩略图
- **文件预览**: 在线预览各类文件
- **存储管理**: 本地存储和对象存储支持
